# Hashtag Strategy (Section 22 of brand-guidelines-v2.md — tiered + rotated)

Don't paste a fixed block. For each post, build a set of **8-15 tags** by mixing
tiers and matching them to THAT post's actual topic. Rotate within each tier so the
account doesn't repeat the same stack every day (Instagram favors varied, relevant
tags and can suppress identical repeated blocks). Prefer placing the tags in the
**first comment** (`hashtags_in_first_comment: true`) to keep captions clean.

A good set ≈ 1-2 branded + 2-3 niche + 2-3 medium + 2-4 topical (today's subject).

## layer8culture

### Branded (always include 1-2)
#Layer8Culture #TechThursday

### Niche / community (2-3 — our people, lower competition, high intent)
#BuildInPublic #AIFluency #FutureBuilders #CreatorTech #AICommunity
#BuildWithAI #LearnInPublic #IndieHackers #AIBuilders

### Medium reach (2-3 — broader but still relevant)
#ArtificialIntelligence #AItools #GenerativeAI #AIagents #TechCreators
#CodingLife #DevCommunity #Automation #PromptEngineering

### Topical (2-4 — swap to match the post's subject)
- Model/news posts: #ChatGPT #Claude #OpenAI #Anthropic #LLM #AInews
- Agentic / automation: #AIagents #AgenticAI #Workflow #NoCode
- Creator systems / build-in-public: #ContentCreation #CreatorEconomy #SideProject
- Tech Thursday / live show: #LiveStream #TechShow #WeeklyShow
- Culture / vision: #Afrofuturism #TechCulture #DigitalCulture

### Avoid
Irrelevant or spammy-broad tags (e.g. #lofi on main-account posts, #love, #viral,
#followforfollow) and banned/oversaturated tags. Don't exceed ~15. Keep every tag
defensibly relevant to the specific post.
